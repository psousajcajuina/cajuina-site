Este é um post de teste para validar a funcionalidade de embeds do YouTube no markdown.

## Como Funciona

Agora é possível adicionar vídeos do YouTube diretamente no conteúdo usando a sintaxe:

`[[youtube:VIDEO_ID]]`

## Conclusão

O recurso facilita a adição de conteúdo multimídia nos posts sem precisar mexer em HTML ou configurações complexas. Os vídeos são responsivos e se adaptam a diferentes tamanhos de tela.

Para adicionar um vídeo, basta copiar o ID do vídeo (parte final da URL do YouTube) e usar o formato `[[youtube:ID_DO_VIDEO]]`.
